# homebrew-tap

Homebrew tap for [tuigram](https://github.com/queq-co/tuigram), a terminal UI
(Ratatui) Telegram client written in Rust.

## Install

```sh
brew install queq-co/tap/tuigram
```

## Update

Formula updates are opened automatically as a PR against this repo whenever a
new `tuigram` release is tagged, then auto-merged once the formula's own test
passes (see `.github/workflows/tests.yml`).
